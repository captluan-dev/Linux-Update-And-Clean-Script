# Linux System Update And Clean Script

Script em Bash para atualização e limpeza automática de sistemas baseados em Debian/Ubuntu.

##  Funcionalidades

- Verifica conexão com a internet
- Verifica compatibilidade do sistema
- Atualiza pacotes
- Remove pacotes desnecessários
- Limpa cache do sistema

##  Requisitos

- Distribuição baseada em Debian/Ubuntu
- Permissão sudo

##  Como usar

```bash
chmod +x update.sh
./update.sh
